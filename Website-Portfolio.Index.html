  • Designed and provisioned scalable, secure AWS infrastructure using Terraform and CloudFormation, enabling repeatable, environment-consistent deployments across Dev, SIT, QA, and Production — reducing environment spin-up time by ~70%.

  • Containerized microservices using Docker and deployed production workloads on Kubernetes (EKS) and Amazon ECS (Fargate) with Helm-managed releases — improving container resource utilization by ~40% and supporting 3× traffic growth without infrastructure changes.

  • Configured real-time observability stack using ELK Stack, Prometheus, Grafana, AWS CloudWatch, CloudTrail, and Splunk — enabling proactive incident detection and reducing MTTR by ~55%.
  • Managed and right-sized EC2 instances, EBS volumes, AMIs, Elastic Load Balancers, and Auto Scaling Groups — reducing monthly EC2 spend by ~25% through Reserved Instance optimization and right-sizing.
  • Architected multi-tier VPCs with subnets, routing tables, NAT gateways, internet gateways, and VPC peering between accounts for secure, segmented cross-account communication.
  • Maintained IAM roles, policies, and least-privilege access controls across all environments, ensuring SOC 2 compliance and audit readiness at all times.
  • Authored Puppet modules and manifests to automate Linux system configuration and manage WebSphere Application Server deployments on Ubuntu production hosts.
  • Developed Python and Bash/Shell automation scripts for operational tooling, backup automation, and disaster recovery procedures — eliminating ~8 hours/week of manual operational overhead.
  • Implemented SonarQube for continuous static code analysis with enforced quality thresholds integrated into Jenkins pipelines, blocking sub-threshold builds from reaching staging.
